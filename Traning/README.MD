#### 20210802 (월)
- 객체지향언어의 특징
1. 캡슐화(Encapsulization)
 클래스(객체)
 접근제한자(Access Modifier)
  public	(무제한 사용/공유)
  protected	(상속: 다른패키지의 상속받은 자식클래스와 공유)
  default	(같은 패키지에서만 공유)
  private	(클래스 내부에서만 공유)

2. 상속(Inheritance)
 클래스의 속성을 다른 클래스가 무려받는 것.
3. 다형성(Polymophism)
 메소드 오버로딩
 생성자 오버로딩
 
4. 추상화

final	상속가능, 재정의불가(필드, 메소드),
		상속불가(클래스)
